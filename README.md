# Samurai Revenge

Samurai Revenge je moj prvy herny projekt vytvoreny v Godote. Je to jednoducha 2D platformova hra so samurajom, questami, nepriatelmi, NPC postavou, portalmi a zakladnym postupom cez viac levelov.

## O hre

- 2D platformer s pohybom, skokom, sprintom a utokom.
- Questy na zbieranie minci, porazenie nepriatelov a dosiahnutie portalu.
- Nepriatelia ako zaby a vcela.
- NPC Jawy zadava ulohy a dava odmenu.
- Hudba pokracuje medzi levelom 2 a levelom 3.

## Ovládanie

- `A` / `D` - pohyb dolava a doprava
- `Space` - skok
- `Shift` - sprint
- `J` - utok
- `E` - interakcia s NPC

## Assety

Projekt pouziva rozne 2D sprity, hudbu a zvuky pocas vyvoja hry. Cast vizualnych assetov pochadza z itch.io asset packov. Tento projekt sluzi hlavne ako ucenie Godotu a tvorby hier.

## Stav projektu

Projekt je vo vyvoji. Kod, levely aj grafika sa este mozu menit.
